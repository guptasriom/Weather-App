# 🌦️ SKYE Weather App

A modern weather web application built using Flask that provides real-time weather updates with an interactive UI.

## 🚀 Live Demo
https://skye-weather-ymf0.onrender.com

## 🛠 Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- OpenWeather API
- Render (Deployment)

## ✨ Features
- Real-time weather data
- Dynamic UI based on weather conditions
- Animated backgrounds (rain, snow, thunder, etc.)
- Responsive design
